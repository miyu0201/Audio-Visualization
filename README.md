# Audio-Visualization

此Demo基于ARFoundation制作，目前仅兼容ARKit，使用URP管线，Unity版本2019.4.12f。

Demo中的音频可视化部分，来自于[Youtube教程](https://www.youtube.com/watch?v=5pmoP1ZOoNs&list=PL3POsQzaCw53p2tA6AWf7_AWgplskR0Vo)，想制作更多效果可以前往学习。
